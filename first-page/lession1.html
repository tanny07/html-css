Lets learn how to create a simple html page

* I am using sublime text software to learn html and css

* Always start with !doctype in the code part of html

* Under !doctype initialize html opening and closing brackets

* Under html initialize the head first and then the body of the page

* In the head you can initialize the title of the page

* CSS styling part can be done with style in the head of html

* Write the content in the body of the page

* At the end just add an image of code to the page
